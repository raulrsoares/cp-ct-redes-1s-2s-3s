Usuários

    COMANDOS
        adduser [nome-do-usuário] - Criar um usuário
        adduser [nome-do-usuário] [nome-do-grupo] - Adicionar um usuário a um grupo

        passwd [nome-do-usuário] - Trocar a senha de um usuário

        id [nome-do-usuário] - Ver informações sobre um usuário

        deluser [nome-do-usuário] - Deletar um usuário
        deluser [nome-do-usuário] [nome-do-grupo] - Retirar um usuário do grupo

    ARQUIVOS
        /etc/passwd - Arquivo que armazena os usuários no Linux
        /etc/shadow - Arquivo que armazena as senhas dos usuários no Linux

    UID - User ID - Identificação do Usuário
